# limpieza-datos-titanic
Data cleaning project using the Titanic dataset from Kaggle

Proyecto de limpieza y preparación del dataset Titanic, orientado a mejorar la calidad de los datos y dejarlos listos para análisis y visualización.
Incluye tratamiento de valores faltantes, estandarización de variables y creación de nuevas características.
El dashboard en Power BI se encuentra en desarrollo.
